# clj-tigase

clj-tigase is a simple library for working with Tigase XMPP servers
from Clojure.

## Using

With Leiningen:

    [clj-tigase "0.1.0-SNAPSHOT"]


With Maven:

    <dependency>
      <groupId>clj-tigase</groupId>
      <artifactId>clj-tigase</artifactId>
      <version>0.1.0-SNAPSHOT</version>
    </dependency>
